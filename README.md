# Airbnb Price Prediction ML Model
<br>
This model predicts Airbnb listing prices using machine learning.
<br>
## Problem Statement <br>
Estimate listing prices based on room type, neighborhood frequency, and booking details.
<br>
## Approach <br>
- Data cleaning & preprocessing <br>
- Log transformation of price <br>
- Feature engineering <br>
- Linear Regression <br>
- Random Forest Regression <br>
<br>
## Results <br>
- R² score ≈ 0.6 <br>
- Model generalizes well on unseen data <br>
<br>
## Tech Stack <br>
- Python <br>
- Pandas, NumPy <br>
- Scikit-learn <br>
- Jupyter Notebook
<br>
## Future Improvements <br>
- Gradient Boosting / XGBoost <br>
- Location-based features <br>
